Sliding Window

When to Use: Subarrays / substrings problems

Time Complexity: O(n)

Template:
int left = 0;
for(int right = 0; right < n; right++){
    while(condition){
        left++;
    }
}

Dynamic Programming

When to Use: Overlapping subproblems

Time Complexity: O(n) or O(nÂ²)

Template:
dp[0] = base;
for(int i = 1; i < n; i++){
    dp[i] = min/max(previous states);
}

Recursion

When to Use: Tree, backtracking, divide & conquer

Time Complexity: Depends on calls

Template:
void solve(){
    if(base_case) return;
    solve(smaller_problem);
}

Queue

When to Use: BFS, level order traversal

Time Complexity: O(1) per operation

Queue q;
q.push(x);
q.pop();

Binary Search

When to Use: Sorted array

Time Complexity: O(log n)

int low = 0, high = n-1;
while(low <= high){
    int mid = (low + high)/2;
}
